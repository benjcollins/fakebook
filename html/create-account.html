What is the Fakebook?

The Fakebook is a project created by Ben Collins in an attempt to create a simple social media platform only using web technologies avaliable at the time when Facebook was founded.

This is in an attempt to demonstrait that you can build good looking responsive websites without the use of bloated libaries like React or Vue. The default empty project in React has a size of a few megabytes before it is filled with any content. Compared to the landing page here which is a single HTML document containing no Javascript or CSS and is only a few Kilobytes in size.
